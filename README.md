# hub-mkdao

## 项目简介

`hub-mkdao` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 页面的集中存储与展示空间。

所有页面均为独立构建，彼此无关联，适用于需要简单托管多份 HTML 内容的场景。

## 目录说明

```
.
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（CSS、图片、字体等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本说明文件
```

- `pages/`：每个子文件夹或文件代表一个独立的 HTML 页面，命名尽量反映页面用途。
- `assets/`：存放多个页面可能共用的样式、脚本或媒体资源，避免重复引用。

## 页面归档说明

本仓库中的 HTML 页面均为独立存档，可能包含但不限于以下类型的内容：

- 静态信息展示页
- 小型工具或演示页面
- 学习笔记或文档页
- 临时项目原型

每个页面均不依赖外部后端服务，可直接在浏览器中打开运行。页面之间无关联，可独立访问。
